## 交互联动示意

```text
用户操作                          系统响应
────────────────────────────────────────────────────────────────────

1. 点击左侧"硬约束冲突: 5"
   │
   ├──> 中间表格: 高亮所有硬约束冲突单元格
   ├──> 左侧冲突列表: 只显示硬约束冲突
   └──> 右侧详情区: 显示硬约束冲突汇总

2. 点击左侧冲突列表中的"张三 连续工作超限"
   │
   ├──> 中间表格: 自动滚动并高亮张三的相关单元格
   ├──> 左侧冲突列表: 高亮该冲突项
   └──> 右侧详情区: 显示该冲突的详细信息和建议操作

3. 点击中间表格中的冲突单元格
   │
   ├──> 左侧冲突列表: 高亮对应的冲突项
   └──> 右侧详情区: 显示该单元格的冲突详情

4. 在右侧详情区点击"解决"按钮
   │
   ├──> 中间表格: 移除冲突标记，更新单元格状态
   ├──> 左侧统计摘要: 更新冲突数量
   ├──> 左侧冲突列表: 移除该冲突项
   └──> 右侧详情区: 显示下一个冲突或关闭
```

## 响应式布局变化

### 大屏幕 (1920px+)
```
┌────────┬──────────────┬────────┐
│  20%   │     55%      │  25%   │
│  左侧  │   主内容区   │  右侧  │
└────────┴──────────────┴────────┘
```

### 中等屏幕 (1366px-1920px)
```
┌────────┬──────────────┬────────┐
│  20%   │     60%      │  20%   │
│  左侧  │   主内容区   │  右侧  │
└────────┴──────────────┴────────┘
```

### 小屏幕 (1024px-1366px)
```
┌────────┬──────────────────────┐
│  15%   │         85%          │

```


## 设计优势

1. **视觉焦点清晰**：中间表格占据最大面积，成为绝对的视觉中心
2. **信息获取高效**：左侧统计摘要让关键问题一目了然
3. **操作流程顺畅**：左侧发现 -> 中间定位 -> 右侧解决，工作流自然
4. **界面整洁**：筛选栏可折叠，减少视觉噪音
5. **扩展性好**：三个区域职责明确，易于扩展新功能
6. **灵活可调**：支持拖拽调整各区域宽度，适应不同使用习惯
